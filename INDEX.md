## 🚀 БЫСТРЫЙ СТАРТ

```bash
# Клонирование репозитория
git clone https://github.com/WE-RAZDOR/RSecure.git
cd RSecure

# Установка зависимостей
pip install -r requirements.txt

# Запуск базовой защиты
python run_rsecure.py

# Запуск с панелью управления
python run_rsecure_with_dashboard.py
```

## 🛡️ КЛЮЧЕВЫЕ СЛОИ ЗАЩИТЫ

### 🧠 **Нейроволновая защита**
- Мониторинг WiFi/Bluetooth через macOS APIs
- Внешние SDR модули (HackRF, RTL-SDR)
- DIY модули (~$975)
- Биометрическая корреляция (ECG, GSR, температурные сенсоры)
- FFT анализ и статистическая валидация

### 🛡️ **Антипозиционирование**
- Защита от WiFi отражений
- Обфускация CSI данных
- Генерация многолучевых помех
- Рандомизация фазы и амплитуды

### 🔓 **DPI Обход**
- Фрагментация пакетов
- TLS SNI Splitting
- Обфускация HTTP заголовков
- Domain Fronting
- Цепочки прокси и VPN
- Tor интеграция
- Протокол мимикрия

### 🧠 **Психологическая защита**
- Мониторинг нейронных весов
- Анализ аудио потоков
- Защита от weight adjustment атак

### 🎥 **Визуальная безопасность**
- Мониторинг мерцаний экрана
- Защита от зрительных атак
- Фильтрация и нормализация

- **ОС**: macOS 10.15+

- **ОС**: macOS 12.0+
- **Python**: 3.11+
- **RAM**: 16GB+

- **SDR**: HackRF One или RTL-SDR
