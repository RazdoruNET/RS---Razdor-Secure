# Настройка браузера для работы с Фин-Шторм прокси

## Запуск прокси сервера

```bash
python3 fin_storm_proxy.py
```

Сервер запустится на `127.0.0.1:8080`

## Настройка браузеров

### Chrome/Edge/Opera
1. Откройте настройки браузера
2. Перейдите в "Система" → "Настройки прокси-сервера"
3. Выберите "Использовать прокси-сервер"
4. Адрес: `127.0.0.1`
5. Порт: `8080`
6. Нажмите "Сохранить"

### Firefox
1. Откройте настройки
2. Перейдите в "Общие" → "Параметры сети"
3. Выберите "Ручная настройка прокси"
4. HTTP прокси: `127.0.0.1` порт `8080`
5. Отметьте "Использовать этот прокси-сервер для всех протоколов"
6. Нажмите "OK"

### Safari (macOS)
1. Откройте "Системные настройки"
2. Перейдите в "Сеть"
3. Выберите активное подключение
4. Нажмите "Дополнительно"
5. Перейдите во вкладку "Прокси"
6. Отметьте "Веб-прокси (HTTP)"
7. Адрес: `127.0.0.1`
8. Порт: `8080`
9. Нажмите "OK" → "Применить"

## Проверка работы

1. Запустите прокси сервер
2. Настройте браузер
3. Откройте [https://www.youtube.com](https://www.youtube.com)
4. Проверьте доступ к другим заблокированным сайтам

## Техники обхода

Прокси использует те же техники, что и модуль Фин-Шторм:
- **TCP Window Scaling = 1** - минимальный размер окна
- **SNI маскировка** - запросы отправляются как от `sbp.nspk.ru`
- **Банковские заголовки** - имитация финансового приложения СБП

## Логирование

Сервер выводит логи всех запросов:
```
🚀 Запускаю Фин-Шторм прокси сервер...
📍 Адрес: 127.0.0.1:8080
⛓️ Техники: TCP Window=1, SNI=sbp.nspk.ru, Banking headers
📋 GET https://www.youtube.com/ - 200
```

## Остановка

Нажмите `Ctrl+C` в терминале с запущенным прокси.
