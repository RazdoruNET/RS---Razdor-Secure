# Proxy Tools

This directory contains all proxy-related scripts and tools for the RS Razdor Secure system.

## Proxy Implementations

### Core Proxy Scripts
- `fin_storm_proxy.py` - Fin Storm proxy implementation
- `enhanced_fin_storm_proxy.py` - Enhanced version of Fin Storm proxy
- `final_proxy.py` - Final proxy implementation
- `robust_proxy.py` - Robust proxy with error handling
- `simple_proxy.py` - Simple proxy implementation
- `simple_working_proxy.py` - Working simple proxy
- `ultimate_proxy.py` - Ultimate proxy with all features

### Specialized Proxies
- `http_tunnel_proxy.py` - HTTP tunneling proxy
- `ngrok_proxy.py` - Ngrok integration proxy
- `system_proxy.py` - System-level proxy
- `system_proxy_manager.py` - System proxy management
- `white_ghost_proxy.py` - White Ghost proxy implementation
- `white_ghost_proxy_fixed.py` - Fixed version of White Ghost proxy
- `working_ngrok_proxy.py` - Working Ngrok proxy

### Integration
- `rsecure/modules/defense/vpn_proxy.py` - VPN proxy module (part of core system)

## Usage

Each proxy script can be run independently or as part of the larger system. Most proxies require configuration files and proper network setup.
