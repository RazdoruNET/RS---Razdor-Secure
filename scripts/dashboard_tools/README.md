# Dashboard Tools

This directory contains all dashboard and monitoring interface scripts for the RS Razdor Secure system.

## Dashboard Implementations

### Main Dashboards
- `advanced_dashboard.py` - Advanced monitoring dashboard
- `optimized_dashboard.py` - Optimized performance dashboard
- `russian_dashboard.py` - Russian language dashboard
- `simple_dashboard.py` - Simple monitoring interface
- `simple_rsecure_dashboard.py` - Basic RSecure dashboard

### Specialized Dashboards
- `turbo_escalation_dashboard.py` - Turbo escalation monitoring
- `turbo_russian_dashboard.py` - Turbo Russian dashboard

### Integration
- `run_rsecure_with_dashboard.py` - Run system with dashboard (moved to startup/)
- `rsecure/utils/dashboard.py` - Core dashboard utilities (part of main system)

## Usage

Dashboards provide real-time monitoring and control interfaces. Most dashboards require the main system to be running and proper authentication.

## Features

- Real-time monitoring
- Threat visualization
- System control interfaces
- Performance metrics
- Security analytics
