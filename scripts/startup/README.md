# Startup Scripts

This directory contains all scripts for starting, launching, and setting up the RS Razdor Secure system.

## Scripts

### Main Application Starters
- `run_rsecure.py` - Main application runner
- `run_rsecure_with_dashboard.py` - Start with dashboard interface
- `run_dpi_bypass_daemon.py` - DPI bypass daemon launcher

### Proxy Starters
- `start_fixed_proxy.py` - Start fixed proxy server
- `start_full_system_proxy.py` - Start full system proxy
- `start_white_ghost.py` - Start White Ghost proxy
- `launch_dpi_bypass_proxy.py` - Launch DPI bypass proxy

### Setup Scripts
- `setup_http_proxy.py` - Configure HTTP proxy settings
- `setup_system_proxy.py` - Configure system proxy settings

### Shell Scripts
- `start_dpi_bypass.sh` - Shell script for DPI bypass
- `start_rsecure.sh` - Shell script for main application

## Usage

Run scripts with appropriate Python interpreter or shell. Most scripts require root privileges for system-level operations.
