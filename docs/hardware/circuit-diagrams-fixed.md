## RF модуль

### 1. HackRF One подключение

```
HackRF One Pinout:
┌─────────────────────────────────────────────────────────┐
│ 1  ANT        ←→ Антенна (SMA connector)                  │
│ 2  TX         ←→ Передача (не используется)                │
│ 3  RX         ←→ Прием (активен)                          │
│ 4  CLK        ←→ Внешний тактовый генератор (опционально) │
│ 5  GND        ←→ Земля                                    │
│ 6  VCC        ←→ Питание 5V (от USB)                      │
│ 7  USB        ←→ Raspberry Pi USB 3.0                     │
│ 8  RESET      ←→ Сброс (не используется)                  │
└─────────────────────────────────────────────────────────┘
```

### 2. RTL-SDR подключение

```
RTL-SDR v3 Pinout:
┌─────────────────────────────────────────────────────────┐
│ 1  ANT        ←→ Антенна (MCX connector)                  │
│ 2  BIAS-T     ←→ Питание антенны 5V (опционально)        │
│ 3  GND        ←→ Земля                                    │
│ 4  USB        ←→ Raspberry Pi USB 2.0                     │
│ 5  PPS        ←→ Pulse Per Second (GPS sync)              │
└─────────────────────────────────────────────────────────┘
```

### 3. RF цепь с фильтрами

```
RF Signal Chain:
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│   Antenna   │───▶│   GDT       │───▶│   BPF       │───▶│   ATT       │
│             │    │ Protection  │    │ 2.4-2.5GHz  │    │ 6dB 50Ω     │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
                                                           │
                                                           ▼
                                                   ┌─────────────┐
                                                   │   SDR       │
                                                   │  Device     │
                                                   └─────────────┘
```

**Компоненты:**
- **GDT**: Gas Discharge Tube (защита от статического электричества)
- **BPF**: Band Pass Filter (фильтр 2.4-2.5 GHz для WiFi)
- **ATT**: Attenuator (аттенюатор 6dB для защиты входа)

## Схемы антенн

### 1. Дипольная антенна (1-100 MHz)

```
Half-Wave Dipole:
┌─────────────────────────────────────────────────────────┐
│                                                         │
│              ┌─────────────┐                           │
│              │   Balun     │                           │
│              │  1:4 (200Ω) │                           │
│              └──────┬──────┘                           │
│                     │                                   │
│         ┌───────────┴───────────┐                       │
│         │                       │                       │
│    ┌────┴────┐              ┌────┴────┐                │
│    │ Element │              │ Element │                │
│    │   λ/4   │              │   λ/4   │                │
│    │ (провод)│              │ (провод)│                │
│    └────┬────┘              └────┬────┘                │
│         │                       │                       │
│         ▼                       ▼                       │
│      ┌─────┐                 ┌─────┐                    │
│      │ 50Ω │                 │ 50Ω │                    │
│      │ Feed │                 │ Feed │                    │
│      └─────┘                 └─────┘                    │
│                                                         │
│  Расчет длины:                                          │
│  L = 143 / f(MHz) метров (каждый элемент)              │
│                                                         │
│  Примеры:                                               │
│  f = 10 MHz → L = 14.3 м                               │
│  f = 50 MHz → L = 2.86 м                               │
│  f = 100 MHz → L = 1.43 м                              │
│                                                         │
└─────────────────────────────────────────────────────────┘
```
