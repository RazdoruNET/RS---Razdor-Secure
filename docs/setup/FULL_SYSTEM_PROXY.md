- 📱 Все приложения (Telegram, Discord, Steam)
- 🎮 Все игры и программы

## 🛠️ РЕШЕНИЕ ПРОБЛЕМ

### ❌ Прокси не работает:
```bash
# 1. Проверьте статус
sudo python3 start_full_system_proxy.py --status

# 2. Проверьте тест
sudo python3 start_full_system_proxy.py --test

# 3. Перезапустите
sudo python3 start_full_system_proxy.py --stop
sudo python3 start_full_system_proxy.py
```

### ❌ Приложения не работают:
1. Перезапустите приложение
2. Проверьте настройки сети в приложении
3. Отключите и включите Wi-Fi

### ❌ Медленная скорость:
1. Попробуйте другой порт:
```bash
sudo python3 start_full_system_proxy.py --port 1081
```
2. Отключите ненужные приложения
3. Перезагрузите компьютер
