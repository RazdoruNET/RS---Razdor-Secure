# Научные основы нейроволновой защиты

## Обзор

Документация представляет научное обоснование методов защиты от воздействия на мозговые волны через беспроводные интерфейсы, основанное на современных исследованиях в области нейрофизиологии, электромагнетизма и кибербезопасности.

## 🧠 Нейрофизиологические основы

### Мозговые волны и их функции

#### Дельта-волны (0.5-4 Hz)
- **Функция**: Глубокий сон, восстановление, регенерация
- **Анатомия**: Таламус, кора головного мозга
- **Исследования**: [Steriade et al., 1993] - Координированная активность во время медленного сна
- **Воздействие**: Нарушение сна, когнитивные нарушения

#### Тета-волны (4-8 Hz)
- **Функция**: Медитация, память, творчество
- **Анатомия**: Гиппокамп, префронтальная кора
- **Исследования**: [Buzsáki, 2002] - Тета-ритмы и когнитивные функции
- **Воздействие**: Влияние на память и обучение

#### Альфа-волны (8-12 Hz)
- **Функция**: Расслабление, фокус внимания
- **Анатомия**: Затылочная кора, сенсорные области
- **Исследования**: [Klimesch, 1999] - Альфа-ритмы и когнитивная обработка
- **Воздействие**: Изменение восприятия и внимания

#### Бета-волны (12-30 Hz)
- **Функция**: Активное бодрствование, решение задач
- **Анатомия**: Префронтальная кора, моторные области
- **Исследования**: [Engel & Fries, 2010] - Бета-ритмы и когнитивный контроль
- **Воздействие**: Влияние на принятие решений

#### Гамма-волны (30-100 Hz)
- **Функция**: Высшая когнитивная активность, сознание
- **Анатомия**: Ассоциативные области коры
- **Исследования**: [Fries, 2009] - Гамма-ритмы и нейронная коммуникация
- **Воздействие**: Модуляция сознания и восприятия

### Механизмы электромагнитного воздействия

#### Ионные каналы и мембранный потенциал
```python
# Математическая модель воздействия на мембранный потенциал
V_m = V_rest + ΔV_em
ΔV_em = k * E_field * cos(2π * f * t + φ)

# где:
# V_m - мембранный потенциал
# V_rest - потенциал покоя (~-70mV)
# E_field - напряженность ЭМ поля
# f - частота воздействия
# k - коэффициент чувствительности
```

#### Нейронная синхронизация
- **Фазовая синхронизация**: Выравнивание фаз нейронной активности
- **Энтрейнмент**: Принуждение нейронов к частоте внешнего стимула
- **Резонансные явления**: Усиление при совпадении частот

## 🛡️ Механизмы защиты

### Электромагнитное экранирование

#### Фарадеевская клетка
```python
# Расчет эффективности экранирования
SE(dB) = 20 * log10(1 + (t/δ))
δ = sqrt(2/(μ*σ*ω))

# где:
# SE - эффективность экранирования
# t - толщина материала
# δ - глубина скин-слоя
# μ - магнитная проницаемость
# σ - электропроводность
# ω - угловая частота
```

#### Материалы экранирования
- **Медь**: Высокая электропроводность, эффективность > 100 dB
- **Алюминий**: Легкий, эффективность ~80 dB
- **Мю-металл**: Высокая магнитная проницаемость, эффективность > 120 dB

### Частотная фильтрация

#### Низкочастотные фильтры
```python
# RC фильтр первого порядка
H(jω) = 1 / (1 + jωRC)
f_c = 1 / (2πRC)

# где:
# H(jω) - передаточная функция
# f_c - частота среза
```

#### Полосовые фильтры
- **Sallen-Key**: Активный фильтр высокого качества
- **Чебышёв**: Резкая переходная характеристика
- **Баттерворт**: Плоская АЧХ в полосе пропускания

### Адаптивные алгоритмы защиты

#### LMS (Least Mean Squares) фильтр
```python
# Адаптивная фильтрация
w[n+1] = w[n] + μ * e[n] * x[n]
e[n] = d[n] - w^T[n] * x[n]

# где:
# w[n] - весовые коэффициенты
# μ - шаг адаптации
# e[n] - ошибка фильтрации
# x[n] - входной сигнал
```

#### Калмановская фильтрация
- **Предсказание**: Оценка состояния системы
- **Коррекция**: Обновление на основе измерений
- **Оптимизация**: Минимизация ошибки оценки

### Программная защита

#### Цифровая обработка сигналов
```python
# Адаптивный фильтр
def adaptive_filter(input_signal, reference_signal, mu=0.01):
    n = len(input_signal)
    w = np.zeros(n)
    y = np.zeros(n)
    e = np.zeros(n)
    
    for i in range(1, n):
        y[i] = np.dot(w[:i], input_signal[:i][::-1])
        e[i] = reference_signal[i] - y[i]
        w[:i] += mu * e[i] * input_signal[:i][::-1]
    
    return y, e
```

#### Машинное обучение
- **Классификация угроз**: Random Forest, SVM
- **Детекция аномалий**: Isolation Forest, One-Class SVM
- **Предсказание**: LSTM, Transformer сети
