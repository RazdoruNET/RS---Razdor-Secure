## 📈 Инфографика: Алгоритмы детекции

```
┌─────────────────────────────────────────────────────────────────┐
│                    АЛГОРИТМЫ ДЕТЕКЦИИ УГРОЗ                    │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  1. АЛГОРИТМ СПЕКТРАЛЬНОГО АНАЛИЗА                            │
│  ┌─────────────────────────────────────────────────────────────┐  │
│  │  Вход: временной ряд (1000 сэмплов)                        │  │
│  │  │                                                           │  │
│  │  Шаг 1: Применение окна Ханна                               │  │
│  │  windowed_signal = signal * hanning(N)                       │  │
│  │                                                             │  │
│  │  Шаг 2: Быстрое преобразование Фурье                        │  │
│  │  FFT_result = fft(windowed_signal)                          │  │
│  │  frequencies = fftfreq(N, 1/fs)                              │  │
│  │  magnitude = abs(FFT_result)                                 │  │
│  │                                                             │  │
│  │  Шаг 3: Анализ мозговых волн                                │  │
│  │  for band in [delta, theta, alpha, beta, gamma]:            │  │
│  │      mask = (frequencies >= band.min) & (frequencies <= band.max)│  │
│  │      band_energy = sum(magnitude[mask])                      │  │
│  │                                                             │  │
│  │  Шаг 4: Детекция аномалий                                   │  │
│  │  z_score = abs(current - baseline) / std(baseline)           │  │
│  │  anomaly = z_score > threshold                              │  │
│  └─────────────────────────────────────────────────────────────┘  │
│                                                                 │
│  2. АЛГОРИТМ КОРРЕЛЯЦИОННОГО АНАЛИЗА                           │
│  ┌─────────────────────────────────────────────────────────────┐  │
│  │  Вход: ЭМ сигнал, биометрические данные                     │  │
│  │                                                             │  │
│  │  Шаг 1: Синхронизация временных рядов                        │  │
│  │  em_aligned = interpolate(em_signal, target_length)         │  │
│  │  bio_aligned = interpolate(bio_signal, target_length)       │  │
│  │                                                             │  │
│  │  Шаг 2: Расчет коэффициентов корреляции                     │  │
│  │  correlation = pearsonr(em_aligned, bio_aligned)            │  │
│  │                                                             │  │
│  │  Шаг 3: Статистическая значимость                           │  │
│  │  p_value = scipy.stats.pearsonr(em_aligned, bio_aligned)[1] │  │
│  │  significant = p_value < 0.05                                │  │
│  │                                                             │  │
│  │  Шаг 4: Комплексная оценка                                   │  │
│  │  threat_score = weighted_sum(correlations)                   │  │
│  └─────────────────────────────────────────────────────────────┘  │
│                                                                 │
│  3. АЛГОРИТМ КЛАССИФИКАЦИИ УГРОЗ                              │
│  ┌─────────────────────────────────────────────────────────────┐  │
│  │  Вход: спектральные признаки, биометрические признаки         │  │
│  │                                                             │  │
│  │  Шаг 1: Извлечение признаков                                │  │
│  │  features = [                                                  │  │
│  │      spectral_peaks, spectral_entropy,                         │  │
│  │      band_powers, correlations,                             │  │
│  │      heart_rate_variability, stress_level                   │  │
│  │  ]                                                           │  │
│  │                                                             │  │
│  │  Шаг 2: Нормализация                                          │  │
│  │  normalized_features = (features - mean) / std              │  │
│  │                                                             │  │
│  │  Шаг 3: Классификация                                        │  │
│  │  probabilities = neural_network.predict(normalized_features)   │  │
│  │  threat_class = argmax(probabilities)                        │  │
│  │                                                             │  │
│  │  Шаг 4: Оценка уверенности                                   │  │
│  │  confidence = max(probabilities)                              │  │
│  │  threat_level = confidence * class_weights[threat_class]     │  │
│  └─────────────────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────────────┘
```
