# Руководство по выбору оборудования для RSecure

## 🎯 Обзор

RSecure поддерживает **три уровня конфигурации** нейроволновой защиты - от бесплатного встроенного режима до профессиональной DIY системы. Это руководство поможет выбрать оптимальную конфигурацию для ваших потребностей и бюджета.

---

## 📊 Сравнение режимов работы

| Характеристика | Встроенный режим | Расширенный режим | Гибридный режим |
|----------------|------------------|-------------------|-----------------|
| **Стоимость** | Бесплатно | $30-300 | $975 |
| **Сложность** | Низкая | Средняя | Высокая |
| **Точность детекции** | 70-80% | 85-95% | 95-99% |
| **Частотный диапазон** | 2.4/5 ГГц | 1 МГц - 6 ГГц | 1 МГц - 6 ГГц |
| **Биометрия** | Нет | Опционально | Полная |
| **Научная валидация** | Базовая | Расширенная | Максимальная |
| **Требуемые лицензии** | Нет | Да (SDR) | Да (SDR) |

---

## 🎛️ Режим 1: Встроенный (Builtin)

### ✅ Преимущества:
- **Бесплатно** - использует существующее оборудование
- **Простая установка** - один скрипт
- **Низкое энергопотребление**
- **Легальное использование** без лицензий

### 🔧 Требования:
- MacBook с macOS 10.14+
- WiFi/Bluetooth адаптеры
- Python 3.11+
- 8GB RAM

### 📋 Возможности:
```
✅ Мониторинг WiFi сетей (en0, en1, awdl0)
✅ Сканирование Bluetooth устройств
✅ Анализ системных метрик (CPU, память, сеть)
✅ Базовая детекция аномалий
✅ Статистический анализ (Z-score)
❌ Спектральный анализ RF
❌ Биометрическая корреляция
❌ Расширенная детекция
```

### 🚀 Быстрый старт:
```bash
# Автоматическая установка и запуск
python rsecure/modules/defense/neural_wave_protection.py --mode builtin

# Или через основной интерфейс
python rsecure_main.py --neural-protection --mode builtin
```

### 🎯 Кому подходит:
- **Начинающие пользователи**
- **Тестирование системы**
- **Базовая защита**
- **Ограниченный бюджет**

---

## 📡 Режим 2: Расширенный (External)

### ✅ Преимущества:
- **Расширенный частотный диапазон** 1 МГц - 6 ГГц
- **Профессиональная точность** детекции
- **Научно-обоснованный анализ**
- **Гибкая конфигурация**

### 🔧 Требования:
- SDR приемник (HackRF One или RTL-SDR)
- Raspberry Pi 4B (рекомендуется)
- Антенны для нужных частот
- Лицензия на радиооборудование

### 💰 Варианты оборудования:

#### Бюджетный вариант (~$150):
```
📡 RTL-SDR v3: $30
🖥️ Raspberry Pi Zero 2W: $15
📡 Dipole антенна: $40
🔌 Блок питания: $15
📦 Корпус и монтаж: $50
```

#### Стандартный вариант (~$400):
```
📡 HackRF One: $300
🖥️ Raspberry Pi 4B (4GB): $75
📡 Log-periodic антенна: $120
🔌 Блок питания 12V: $25
📦 Корпус и монтаж: $80
```

### 📋 Возможности:
```
✅ Спектральный анализ 1 МГц - 6 ГГц
✅ FFT анализ с окном Ханна
✅ Детекция RF аномалий
✅ Статистическая валидация
✅ Научные пороги (Z-score > 2.5)
❌ Биометрическая корреляция
❌ Встроенный мониторинг
❌ Гибридный анализ
```

### 🚀 Быстрый старт:
```bash
# Проверка SDR оборудования
python scripts/configure_hardware.py

# Запуск с внешними модулями
python rsecure/modules/defense/neural_wave_protection.py --mode external

# С конфигурацией
python run_hybrid_neural_protection.py --config config/external_only.json
```

### ⚠️ Юридические аспекты:
- **Требуется лицензия** на SDR оборудование в большинстве стран
- **Ограничения мощности** передачи (только прием)
- **Запрещенные частоты** (военные, авиационные)
- **Экранированное помещение** рекомендуется

### 🎯 Кому подходит:
- **Исследователи**
- **RF инженеры**
- **Специалисты по безопасности**
- **Университетские лаборатории**

## 🎯 Рекомендации по выбору

### Для начинающих:
**Начните со встроенного режима** - он бесплатный и легальный. Познакомьтесь с базовыми функциями, прежде чем инвестировать в оборудование.

### Для исследователей:
**Расширенный режим с HackRF One** обеспечивает профессиональные возможности по разумной цене. Добавьте биометрию позже для полной системы.

### Для профессионалов:
**Гибридный режим** - максимальная защита для критических систем. Инвестиции оправданы точностью и научной обоснованностью.

### Для образовательных учреждений:
**Комбинированный подход** - начните с RTL-SDR для групп, добавьте HackRF для продвинутых курсов.
