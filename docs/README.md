## Технический стек

- **Python 3.11+** - Основной язык разработки
- **TensorFlow 2.10+** - Нейросетевой анализ (опционально)
- **Scikit-learn** - ML алгоритмы
- **Scapy** - Анализ сетевых пакетов
- **Flask** - Веб-интерфейс
- **Ollama** - LLM интеграция (опционально)

## Уровни системы

1. **Сбор данных** - Мониторинг системы, сети, процессов
2. **Анализ** - Нейросетевой и ML анализ
3. **Детекция** - Обнаружение угроз и аномалий
4. **Защита** - Активные меры противодействия
5. **Управление** - Корреляция и принятие решений
