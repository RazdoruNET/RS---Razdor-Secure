# Performance Tests

This directory contains performance and load testing scripts for the RS Razdor Secure system.

## Test Categories

### Performance Benchmarks
- Performance testing for proxy implementations
- Load testing for dashboard interfaces
- Memory and CPU usage monitoring
- Network throughput testing

## Usage

Run performance tests with:
```bash
python -m pytest tests/performance/
```

Performance tests help identify bottlenecks and ensure the system can handle expected loads.
